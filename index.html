    CSS syntax and selection

    CSS syntax

    A CSS releset/rules is composed of a selectors and a declaration block
    div{
    background-color: black;
    width: 200px;
    height: 200px;
    }

    The declaration block are grouped in block.Each declaration block coontains CSS declaration which are wrapped by an opening cury brace and a closing curly brace.CSS declaration has to be separated by a semicolon;otherwise the code won't work.It allowed not to put a semicolon in the last declaration.

    CSS Selectors

    The following are simple CSS selectors and there are more CSS selectors lessons as you go through our lesson

    Element Selectors

    It is the simplest way to target all of given elements

    Example

    h1{
    font-weight: bold;
    }

    Class Selector

    Multiple elements can have the same class values.CSS class selectors are very useful when targeting different element.The class selector consists of a dot(.) followed by class name.

    Example


    .underlined{
    font-decoration: underlined;
    }
    .emboldened{
    font-decoration: bold
    }

    ID Selector

    The ID selector is very useful when targeting a single element.An ID selector consists of a dash(#) followed by an ID name.An ID name is defined using the id attribute

    EXmple

    #red{
    background-color: red;
    width: 200px;
    height: 200px;
    }

    Grouping Selectiors

    Sometimes multiple CSS ruleset may have similar declarations.Instant of repeating the same declaration we can simply have a group of selectors in a CSS ruleset.A grouping selector consists of selectors separated by commas.

    Example

    h1,p{
    font-family: times new roman;
    color: green;
    }

    CSS Color

    CSS colors are commonly specified using the following:

    • Color name
    • Hexadecimal color value
    • RGB color value

    Color name

    It defines any color by its own name

    Example

    #color-name{
    color: maroon;
    }

    Hexadecimal color value

    Also known as hex color value.It defines any valid color by a hash followed by thw latters and numbers.

    Example

    #hex-color{
    color: #808000;
    }

    RGB color value

    RGB stand for Red,Green,Blue.It defines any vaid color using the rgb() CSS function.The numer value should only be from 0 to 255.The number represent the intensity of the red,green and blue colors in that particular order

    Example

    #rgb-color{
    color: rgb(0,0,255);
    }

    Background image

    The background-image property set one or more images as abackground of an element.The format of its value should be url("image.jpg").

    Example

    body{
    background-image: url("image.jpg");
    }

    Background image repeat

    CSS automatically reset background image horizontally and vertically.But we can manualy repeat background image horizintally and vertically by using the background-repeat property.The no-repeat value of the property background repeat stops a background image from repeating

    Examples

    Horizontal repeating

    body{
    background-image: url("image.jpg");
    background-repeat: repeat-x;
    }

    Vertical repeating

    body{
    background-image: url("image.jpg");
    background-repeat: repeat-y;
    }

    No repeat

    body{
    background-image: url("image.jpg");
    background-repeat: no-repeat;
    }

    Background-attachment

    The background attachment property sets whwther a background image fixed within the viewport,or scrolls with its containig block

    Valid values

    • scroll: background is fixed relative to the element itself and does not scroll with its content
    • fixed: background is fixed relative to the element contents.If the element has a a scrolling mechanism, the background does not move with element.
    • local: background is fixed relative to the element content.if the element has a scrolling mechanism,the background scrolls with the element content.
    • scroll,local
    • local,scroll

    CSS Margin

    CSS margin create space around or outside element.The following properties set the length of a margin on each side

    • margin-top:sets the margin area on top of an elelment
    • margin-right:sets the margin area on the right side of an element
    • margin-bottom:sets themargin area on tne top of an element
    • margin-left:sets the margin area on the left side of an element

    valid values

    • length
    • percentage
    • auto

    the margin property is shorthand for all margin property and it can have one,two,three or four values

    • When one value is specified, it applies the same gargin to all four sides of an elelment
    • When two values are specified, the first value applies to the top and bottom,the secornd value applies to left and right
    • When three values are specified,the first applied to the top,the second value applies to the left and right,the third value applies to the bottom
    • When four values are specified,first value applies to the top,second value applies to the right,the third value applies to the bottom and the forth value applies to the left

    CSS Padding

    CSS padding create space within an element.It clears an area around the inside of an element

    The following property set the length of the padding on each side of the element
    • padding-top:sets the padding area on the top of an element
    • padding-right:sets the padding area on the right side of an elelment
    • padding-left:sets the padding area on the left sde of an element
    • padding-bottom:sets the padding area on the bottom of an element

    CSS padding shorthand

    The padding CSS property can have one,two,three,or four values

    • Whe one value is specified, it applies to all four sides of an element.
    • When two value specified,first value applies to the top and bottom sides of an element and second value applies to the left and left sides of an element
    • when three value specified,first value applies to the top of an element,second value applies to the left and right of an element and third value applies to the bottom side of an element
    • when four values specified,first value applies to the top side,second value applies to the right side,third value applies to the bottom side and four value applies to the left side of an element

    CSS box models

    CSS box is the foundation of design and layout of the web.In CSS box model,each element representd as a box with the following properties

    • Countent: it is the area where the content of the box is displayed.
    • padding: The space or area between and inner edge of the border,its is transparent
    • Border: the area between margin and the padding,its width,style and color can be changed
    • margin: the area outside the border and its transparent
